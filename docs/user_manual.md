# 사용자 매뉴얼

새로 추가된 공지사항, FAQ/기능 요청, 마이페이지 사용량 화면 안내입니다.

## 채팅 화면

- `/chat` 좌측 하단에 공지사항 미리보기와 최근 30일 LLM 사용량 카드가 표시됩니다.
- 사용량 카드는 개인 기준 요청 수, 총 토큰, USD 비용, 현재 USD/KRW 기준 원화 환산 금액을 보여줍니다.
- 환율 API 조회가 실패하면 `.env`의 `USD_KRW_FALLBACK_RATE` 값을 사용합니다.

## 공지사항

- 위치: `/notices`
- 모든 사용자는 전사 공지와 본인 팀 공지를 볼 수 있습니다.
- 팀장 이상은 공지를 작성하고 상단 고정 또는 보관 처리할 수 있습니다.
- 공지는 `notices` 테이블에 저장됩니다.

## FAQ / 기능 요청

- 위치: `/faq`
- 모든 사용자는 기능 요청이나 질문을 작성할 수 있습니다.
- 상태는 `접수`, `검토`, `반영 예정`, `답변 완료`, `종료`로 관리됩니다.
- 팀장 이상은 답변과 상태를 관리할 수 있고, 작성자는 본인 글을 삭제할 수 있습니다.
- 챗봇별 FAQ(`/chatbots/{id}/faqs`)와 별도로, 이 화면은 전역 기능 요구사항 게시판입니다.

## 마이페이지

- 위치: `/mypage`
- 계정 정보와 개인 LLM 사용량을 확인합니다.
- 기간 필터는 24시간, 7일, 30일, 이번 달 기준을 제공합니다.
- 비용은 `usage_events.cost_usd` 합계 기반의 운영 참고용 추정치입니다.

## 관리자 참고

- 새 테이블:
  - `notices`: 공지 제목, 요약, 본문, 고정 여부, 활성 여부, 작성자, 팀 범위
  - `faq_posts`: 기능 요청/질문, 상태, 관리자 답변, 작성자, 답변자, 팀 범위
- 개인 사용량은 기존 `usage_events`를 재사용합니다.
