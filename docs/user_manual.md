# 사용자 매뉴얼 (운영)

장금상선 그룹 챗봇 **운영 서비스**의 사용자 안내서입니다. 운영 서비스는 **팀 단위로 격리된 문서 RAG 챗봇**으로, 아래 4가지 기능을 제공합니다.

> 워크플로·스킬·도구 연결·Claude Code 같은 실험 기능은 **개발 환경 전용**이며 운영 서비스에는 없습니다(ADR-0009). 개발 기능 문서는 "개발자 문서" 섹션을 참고하세요.

## 1. 채팅 — 질문하고 답 받기

- 위치: `/chat` (로그인 후 첫 화면)
- 입력창에 질문을 적고 Enter 로 전송합니다. Shift+Enter 는 줄바꿈, `/` 입력 시 빠른 명령 메뉴가 열립니다.
- 답변은 실시간 스트리밍으로 표시되고, 생성 중에는 중지(■) 버튼으로 끊을 수 있습니다 — 그때까지 받은 내용은 보존됩니다.
- 회사 문서 기반 답변이 필요하면 입력창 아래 **‘📄 회사 문서로 답하기’** 토글을 켜세요. 답변에 `[출처N]` 뱃지가 붙고, 클릭하면 원문 문서 뷰어가 열립니다.
- 좌측 사이드바: 대화 목록(검색·이름 변경·삭제), 새 채팅. 대화 옆 색 점은 마지막 실행 상태(초록=성공, 빨강=실패)입니다.
- 전송이 실패하면 화면 중앙에 복구 안내가 뜹니다 — 입력한 메시지는 보존되며 즉시 재시도/다른 모델/새 채팅을 선택할 수 있습니다.

## 2. 자료 올리기 — 문서 업로드

- 위치: `/documents`
- PDF · DOCX · PPTX · XLSX · CSV · HWP · MD · TXT 를 드래그앤드롭 또는 클릭으로 업로드합니다.
  - 한 번에 최대 30개 / 파일당 50MB / 합계 150MB.
- 업로드 시 **공개 범위**를 선택합니다: `개인`(나만 — 기본값·권장) / `팀`(팀원 모두 검색·열람). 개인 문서는 목록에서도 다른 팀원에게 보이지 않습니다.
- ⚠️ **팀 문서는 신중하게** — 팀 문서는 팀 전원의 AI 답변에 바로 반영됩니다. 불필요하거나 부정확한 문서(초안·구버전·개인 메모)를 팀으로 올리면 **팀 전체 데이터가 오염되어 잘못된 답변**이 나갈 수 있습니다. 검증된 공식 자료(최신 규정·매뉴얼·공지)만 팀 문서로 올리고, 그 외에는 개인 문서를 사용하세요.
- 폴더로 정리할 수 있고(좌측 폴더 트리, 드래그 이동), 파일별 설명은 업로드 후 자동 요약으로 채워지며 직접 수정할 수 있습니다.
- 업로드 직후 자동으로 검색 인덱싱(청크·임베딩)이 진행됩니다 — 목록의 ‘처리 중’ 배지가 사라지면 챗봇이 인용할 수 있습니다. 실패 시 배지에 사유가 표시됩니다.

## 3. 챗봇 만들기 — 반복 업무를 봇으로

- 위치: `/chatbots`
- 이름·설명·시스템 프롬프트(역할 지시)·모델을 정해 팀 전용 챗봇을 만듭니다.
- **가시성**: `나만` / `우리 팀` / `선택 팀`. 팀 가시성 챗봇은 팀원 모두 채팅 화면에서 선택해 사용할 수 있습니다.
- **문서 연결**: 챗봇에 RAG 문서 묶음을 지정하면 그 챗봇은 항상 해당 문서를 근거로 답합니다.
- 챗봇별 **Q&A 게시판**(`/chatbots/{id}/faqs`): 사용자가 질문을 남기면 담당자 답변 또는 AI 자동 답글(문서 인용 포함)을 받을 수 있습니다.

## 4. 문서 검색 (RAG) — 어떻게 동작하나

- 질문하면 팀 문서에서 의미(벡터)·키워드 검색을 동시에 수행해 관련 청크를 찾고, 그 근거로 답변을 생성합니다.
- 권한은 항상 강제됩니다 — **다른 팀 문서, 다른 사람의 개인 문서는 검색 대상에서 제외**됩니다.
- 답변의 `[출처N]` 을 클릭하면 해당 문서의 원문 위치로 이동합니다. 출처가 없는 일반 지식 답변과 구분해 주세요.

## 계정

- 가입: `/register` — 초대 코드(팀장 발급) 입력 → 승인 후 사용. 로그인 세션은 7일 유지되며 만료 시 자동으로 로그인 화면으로 돌아갑니다.
- 비밀번호·이름 변경: `/mypage`.

## 관리자 참고

- 팀장(team_admin)은 공지 작성, FAQ 답변/상태 관리, 팀 멤버 승인·역할 관리(`/admin`), 팀 활동 기록(`/team`)을 사용할 수 있습니다.
- 저장 용량 쿼터는 `/admin` 의 팀 설정에서 관리합니다(GB 단위).
